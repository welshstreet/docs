# LLC Formation for Welsh Street
## Liability Protection Analysis

---

## Recommendation: Form LLC

**Why**: Immutable contracts don't eliminate frontend operator liability. LLC costs $900-$2,600/year vs $20M+ personal exposure.

**Key Issues**:
- OFAC: Frontend operators liable despite immutable contracts (Tornado Cash precedent)
- MSB: Classification unclear; personal liability if required
- Consumer Claims: Users can sue for losses regardless of contract immutability

---

## Liability Exposure

### Personal Risk (No LLC)
- **OFAC**: $20M civil + $1M criminal + 20 years imprisonment
- **MSB**: $250k federal + state penalties + 5 years
- **Consumer Lawsuits**: All personal assets at risk
- **Tax Reporting**: $280+ per violation

### LLC Protection
- Personal assets shielded
- Lawsuits directed at entity
- Regulatory credibility
- Professional operation

---

## Cost Analysis

**Delaware LLC**:
- Formation: $419 (LegalZoom)
- Annual: $900-$2,600 (franchise tax, agent, tax prep)
- Strong case law, Court of Chancery

**Puerto Rico LLC**:
- Formation: $650-$1,650
- Annual: $1,000-$2,500
- Local jurisdiction if residing in PR

**ROI**: $900-$2,600/year protects against $20M+ exposure = immediate break-even

## Formation Checklist

### 1. Form LLC
- [ ] Choose jurisdiction (Delaware or PR)
- [ ] File via LegalZoom ($329 + $90 state fee)
- [ ] Obtain EIN from IRS
- [ ] Create Operating Agreement
- [ ] Save Certificate of Formation

### 2. Transfer Assets
- [ ] Domain → LLC ownership (update WHOIS)
- [ ] GitHub → Create org, transfer repos
- [ ] Vercel → Create team, transfer projects
- [ ] Supabase → Create org, transfer database
- [ ] Email → Google Workspace (optional)

### 3. Update Documentation
- [ ] Terms.mdx → "Welsh Street, LLC"
- [ ] Privacy.mdx → LLC as data controller
- [ ] Disclaimer.mdx → LLC entity
- [ ] Footer → © 2026 Welsh Street, LLC

### 4. Verify
- [ ] Test all services work
- [ ] Screenshot WHOIS, OFAC firewall
- [ ] Save formation docs
- [ ] Document transfer dates
